The best Part!

So these things were rendered using Blender, an open source free rendering/animation/everything engine. I rendered this using the default Cycles Renderer built into Blender.

Now if you look really closely, you'll see that there are artifacts that exist in the image. It was originally a 2840x2160px image, around a whopping 18.6mb. I then downsampled the image to what you see there, a sad 1280x720px. I think that's a 33.3% reduction in width || height but a 11.1% reduction in area or width && height, this reduced the noise in the image however it still exists, but quite obviously reduced the size significantly.
